bool TaskManagerApp::Services::TaskService::RemoveTask ( string title)
inline

This method removes task from list.

The method searches for the first task with a matching title and removes it. If the title is null, an exception is thrown. If no task is found, it returns false.

Parameters
titleTitle of task that needs to be removed.
Exceptions
ArgumentNullExceptionThrown when task title is null.
Returns
True if the task was found and successfully removed, otherwise returns false.
